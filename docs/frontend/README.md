# Frontend Folder Guide

This folder contains only the web client.

## Location
- App root: `frontend/`

## Tech stack
- React
- Vite

## Commands
From repository root:

```bash
npm run dev:frontend
npm run build:frontend
npm run lint:frontend
npm run preview:frontend
```

## Scope boundary
- UI code, routes, client-side assets, and frontend configs must stay in `frontend/`.
- Do not place backend API/server logic inside this folder.
